Celebal Technologies Internship Task
------------------------------------

Task: Create a User Registration Form using HTML and CSS

Objective:
----------
To build a responsive and user-friendly registration form layout using appropriate HTML input types and styled with CSS using a horizontal two-column layout.

Form Fields Included:
---------------------
1. Username        - <input type="text">
2. Email           - <input type="email">
3. Password        - <input type="password">
4. Birthdate       - <input type="date">
5. Address         - <textarea>
6. Phone Number    - <input type="tel">

Buttons:
--------
- Submit (Saves the form data - currently no backend)
- Cancel (Resets all the fields)

Key Features:
-------------
- Responsive two-column horizontal layout using CSS Grid
- Centered and styled form with shadows and spacing
- Phone number field uses 10-digit pattern for validation
- Fully semantic and accessible HTML
- Separate CSS file (style.css) for styling
